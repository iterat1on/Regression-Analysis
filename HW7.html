1 - (2)

The plot above have shown some kind of positive autocorrelation. So we can define a hypothesis test based on the Runs test.

Let \(P_{++}\) = Prob(\(e_i>0\)|\(e_{i-1}>0\)),
\(P_{-+}\) = Prob(\(e_i>0\)|\(e_{i-1}<0\)).

1. One-sided hypothesis test

\(H_0 : P_{++} = P_{-+}\) (Independent)

\(H_1 : P_{++} > P_{-+}\) (Positive correlation)

2. Two-sided hypothesis test

\(H_0 : P_{++} = P_{-+}\) (Independent)

\(H_1 : P_{++} \neq P_{-+}\) (Correlation exist)

1 - (3)

Runs test

library(randtests) # load library

1. One-sided hypothesis test

runs.test(lm$residuals, alternative = 'left.sided', threshold=0, plot=TRUE)

## 
##  Runs Test
## 
## data:  lm$residuals
## statistic = -3.0366, runs = 6, n1 = 14, n2 = 11, n = 25, p-value =
## 0.001196
## alternative hypothesis: trend

As the result of the Runs test, runs=6, and the plot showed positive correlation.

p-value = 0.001196 < 0.01(\(\alpha\)), thus reject \(H_0\).
Accept \(H_1\) : \(P_{++} > P_{-+}\).(Positive correlation).

2. Two-sided hypothesis test

runs.test(lm$residuals, alternative = 'two.sided',threshold=0, plot=TRUE)

## 
##  Runs Test
## 
## data:  lm$residuals
## statistic = -3.0366, runs = 6, n1 = 14, n2 = 11, n = 25, p-value =
## 0.002393
## alternative hypothesis: nonrandomness

As the result of the Runs test, runs=6, and the plot showed positive correlation.

Actually, the difference between one-sided, two-sided result is just p-value.

Thus we might notice p-value was so small, and it will reject \(H_0\).

p-value = 0.002393 < 0.01 (\(\alpha\)), Reject \(H_0\).

Accept \(H_1 : P_{++} \neq P_{-+}\).(Correlation exists).

3. One-sided hypothesis test

\(H_0 : \rho = 0\) (\(d\) > \(d_U\))

\(H_1 : \rho > 0\) (\(d\) < \(d_L\)), \(\alpha=0.05\)

Reject \(H_0\), if (\(d\) < \(d_L\))

4. Two-sided hypothesis test

\(H_0 : \rho = 0\) (\(d_U\) < \(d\) < 4-\(d_U\))

\(H_1 : \rho \neq 0\) (\(d\) < \(d_L\) or \(d\) > 4-\(d_L\)), \(\alpha=0.1\)

Reject \(H_0\) if, (\(d\) < \(d_L\) or \(d\) > 4-\(d_L\))

\(d\) < \(d_L\), Reject \(H_0\).

Accept \(H_1 : \rho \neq 0\).
